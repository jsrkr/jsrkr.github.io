Dashboard Notes

Proxies

The dashboard distinguishes direct measurement from proxy measurement because digital life is not observed equally well across concepts and geographies.

GenAI search attention

Built from state Google Trends interest in a small GenAI keyword basket. It measures relative search attention, not direct usage or minutes.

Digital dating search attention

Built from a dating keyword basket. Relative search attention, not direct dating-app usage.

Digital media minutes

The narrow ATUS measure uses activity code 120308 for leisure computer or internet use excluding games. This is the cleanest public proxy for online leisure, but it is broader than social media alone.

Screen leisure minutes

The broad ATUS screen-leisure measure adds 120308, 120303, and 120307. It captures leisure computer use, television or movies, and games, so it should be read as broad screen time rather than pure social media.

In-person social minutes

Measured from ATUS socializing and communicating activities. State-year values are used directly only when the ATUS sample is large enough; otherwise the dashboard falls back to pooled or broader geography estimates.

Remote-work time saved

Built directly from ACS remote-work share and ACS one-way commute minutes. The dashboard reports the round-trip daily time saved implied by working from home more often.
