Dashboard Notes

Data Sources

Historical inputs combine fertility, population, ACS, ATUS, and Google Trends-based search-attention measures. Future values are scenario paths, not causal forecasts.

IPUMS ACS 2014-2024

Used for observed state-level remote-work share, employed-worker counts, and one-way commute minutes where the local extract contains commute-time variables. In the current repository build, missing commute-time fields are flagged and replaced with documented region or national fallback inputs rather than a silent flat default.

CDC fertility tables

General fertility rate for women ages 15-44 and total fertility rate by state and year, 2016-2023.

Open CDC fertility source

FRED state population series

Used to build state population totals and annual population growth.

Open FRED population release

ATUS 2016-2023

Used as the time-use benchmark for digital media minutes, broad screen leisure, in-person social time, work at home, work away from home, household work, and unpaid care. State-year estimates are pooled or smoothed when sample sizes are small.

Google Trends

Used only for state-by-year search-attention proxies for GenAI and online dating. Relative search attention, not direct dating-app usage.

Hansen remote-job-posting workbook

Used in the broader pipeline only as an auxiliary signal and fallback reference outside the ACS-backed 2014-2024 window.

WFH Map reference
