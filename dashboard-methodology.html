Dashboard Notes

Methodology

The dashboard combines observed state trends with transparent scenario paths so users can compare alternative digital futures without treating them as causal forecasts.

Observed state layer

For each state, the dashboard stores ACS remote-work share, commute inputs, remote-work time saved, fertility measures, population growth, and other direct state-year inputs for 2014-2024. When the local ACS extract does not contain commute-time microdata, the pipeline flags the fallback and uses the best available state, region, or national substitute.

Observed national time-use layer

ATUS provides the benchmark for digital media minutes, broad screen leisure, in-person social minutes, work at home, work away, household work, and unpaid care.

Digital-media and screen-leisure scenarios use these ATUS-based screen-leisure and digital-media time measures. Some broader digital-access measures are not currently used in the public scenario adjustment because they are not consistently populated across states and years.

Reference path

The state fertility projection begins with a reference path anchored to the last observed general fertility rate and extended using the state's observed trend.

Scenario path

Each public scenario path is defined as reference path plus a transparent scenario adjustment. The remote-work scenario uses the change in remote-work time saved relative to the reference path and is anchored to a documented CPS remote-work fertility benchmark; screen scenarios use ATUS-based digital and screen-leisure minutes; care scenarios use ATUS-based household and unpaid-care minutes.

Projection map

The main map shows scenario effect vs reference at the selected horizon, using diverging colors centered at zero.

Interpretation rule

The central interpretation question is what the marginal digital hour is replacing. A digital hour replacing commuting may have a different fertility implication than a digital hour replacing face-to-face social life.
